# Avatar WebChat

###About
Avatar WebChat is a chatproject by 5 student @ Linnaeus University in Kalmar, Sweden in course 2DV612.

The system is built in following languages

#### Backend
   * Java (Maven)
   * Spring MVC
   * MySQL (Database)  

#### Frontend
   * WebSocket
   * HTML5
   * CSS3
   * Materialize (Framework in CSS)
