# Avatar WebChat

###About
Avatar WebChat is a chatproject by 5 student @ Linnaeus University in Kalmar, Sweden in course 2DV612.

The system i build in following languages

* Java
* Maven (Framework in Java)
* SQL (Database)
* WebSocket
* HTML
* CSS
* Materialize (Framework in CSS)
